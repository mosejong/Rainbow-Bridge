# 라운드2 가이드 — ElevenLabs 한국어 음성 받기 (vs Google)

> 목적: 라운드1은 EL이 **영어권 보이스**라 한국어가 어색했음(핸디캡). 이번엔 **EL 한국어 보이스**로
> 제대로 만들어 Google과 공정 비교한다. **결제 없이** 웹에서 받을 수 있다.
> 본인이 할 일은 딱 둘 — **① mp3 받기 ② 듣고 고르기.** 나머지(집계·결론·문서)는 전부 클로드가 한다.

## 🛟 먼저 — 마음 편히 (꼭)

- 이 평가로 **잘못된 결론이 팀에 가지 않습니다.** 클로드가 애매한 건 "애매함"으로만 보고하고,
  엔진 확정은 사람 점수가 **명확할 때만** 합니다. **느낀 대로만 고르면 되고, 틀릴 수가 없습니다.**
- 자신 없으면 **다시 들어도 됩니다.** 모르겠으면 "비슷"이 정답일 수 있습니다.

## 1. ElevenLabs 한국어 mp3 받기 (클릭 단위 — 2026-06-07 실제로 통한 경로)

1. [elevenlabs.io](https://elevenlabs.io) 접속 → 가입/로그인(구글 계정 가능, 무료).
   - ⚠️ **크롬 자동번역을 끄세요**(우클릭 → "원본 표시"). 번역이 켜져 있으면 메뉴가 "면·약·의"처럼 깨지고 **스튜디오가 "클라이언트측 오류"로 죽습니다.**
2. 왼쪽 메뉴 **Text to Speech**(번역 시 "텍스트 음성 변환") 클릭. 없으면 **Studio**(스튜디오).
   - 💡 처음 화면이 **API 대시보드**(잔액·로그 보임)면 거긴 개발자용 — 위 메뉴로 가야 음성 만듦.
3. 오른쪽 설정 패널에서:
   - **Model(모델)** = **`Eleven Multilingual v2`** (한국어 발음에 필요. 배너의 "v3 써보세요"는 알파판이라 **무시**).
   - **Voice(음성)** 칸 클릭 → 검색창에 **`Korean`** → 따뜻·차분한 보이스 선택. **Voice Library에서 미리 `Add` 안 해도 여기서 바로 검색·선택됨**(Add는 선택사항).
     - 추모라 발랄/광고 톤 X. (실제로 통한 보이스: `Bomisori`·`Annie`·`Esther`·`Jason` 등)
4. 가운데 큰 입력칸에 위 **공통 멘트** 붙여넣기.
5. 슬라이더(속도·안정성 등)는 **기본값 그대로** 두기.
6. 아래 **Generate(생성하기)** → 잠깐 기다림 → 오른쪽 아래 **Download(↓)** 로 mp3 저장.
7. 보이스 바꿔가며 3~6 반복. 받은 파일 정리:
   - 파일명: **`el_ko_보이스이름.mp3`** (예: `el_ko_bomisori.mp3`)
   - 위치: **`ai/tts/_output/compare/`** 폴더.
   - 💡 여러 개 받는 이유: 같은 EL이 여러 번 이기면 우연 아님 → 한 번 청취로도 신뢰 ↑.

> 💡 무료 크레딧으로 짧은 문장 몇 개는 충분합니다("잔액 0달러"는 결제잔액이라 무관). 크레딧 모자라면 1~2종만 받아도 OK.
> **어디서든 막히면 그 화면 캡처해서 나한테 보여주세요.** 어느 버튼인지 짚어드립니다.

## 2. A/B 페어 다시 만들기 (클로드가 안내, 명령 한 줄)

mp3를 `_output/compare/`에 다 넣었으면 터미널에서:

```
# ⚠️ 먼저: 라운드1 PAIRS_SHEET.md 백업하고 싶으면 복사해두기(--force가 덮어씀)
python -m ai.tts.pairwise_listen --force
```

→ 새로 받은 **EL 한국어 보이스 각각**이 **현행 Google warm**과 1:1 페어로 묶입니다(좌우 섞여서).

---

## 3. 듣고 고르기 → 나한테 주기

1. `_output/pairs/` 폴더에서 `pair_01_A` ↔ `pair_01_B` 듣고 → **더 위로되는 쪽** 한 칸(`A`/`비슷`/`B`).
2. `PAIRS_SHEET.md`에 적기, 끝까지.
3. 다 채우면 `_PAIRS_KEY.json` + 표를 **나한테 붙여넣기.**
4. → 내가 **명확한 승 / 무승부 / 구분 불가**로 정직하게 집계하고, 결론·문서·메모리·커밋까지 정리합니다.

> 다시 말하지만 본인은 **받기 + 고르기**만. 판단이 애매하면 그대로 "비슷". 그게 정상이고 결론입니다.
