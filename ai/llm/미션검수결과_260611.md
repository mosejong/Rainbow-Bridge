## 2. gentle (무기력 단계 · 부담 0)

| before (제목 / 카테고리) | after (제목 / 카테고리) | 사유 |
|---|---|---|
| 창문 열고 바람 쐬기 / rest | **가슴에 손 얹고 다독이기** / rest | 🌧️+🪞 날씨 의존 제거 + 자기자비 |
| 창가에서 볕 쬐기 / rest | **창가에 앉아 있기** / rest | 🌧️ "볕"(맑은 날 의존) → 날씨 무관 |
| 아이 사진에 인사 / remembrance | **나에게 한마디 건네기** / rest | 🪞+♻️ 사진 미션 3개 몰림 해소 + 자기자비 |
| 가족에게 짧은 메시지 / connection | **가까운 사람에게 짧은 메시지** / connection | 🤝 일반화 |

|---|---|---|
