## 0. 벡터 DB가 뭐 하는 건데? (먼저 이해)

우리 서비스 예시:

```
보호자: "우리 강아지랑 산책하던 게 너무 그리워요"
        ↓
이 문장을 숫자 묶음(벡터)으로 변환   [0.21, -0.5, 0.8, ...]
        ↓
저장해둔 위로 글·추모 메시지 수천 개 중
숫자가 비슷한 것 = 의미가 비슷한 것 찾기
        ↓
"산책의 추억" 관련 위로 글 4개 꺼냄 → LLM이 답변 생성
```

**벡터 DB = "의미 비슷한 거 빠르게 찾기" 저장소.**
아래 3가지 다 이 일을 함. 차이는 *어디 두고, 누가 관리하고, 얼마 드냐.*

## 1. 한 줄 비유

| | 비유 | 뜻 |
|---|------|-----|
| **ChromaDB** | 내 방 책장 | 내 PC/홈서버에 직접 둠. 공짜. 내가 관리 |
| **MongoDB Atlas** | 쓰던 창고에 칸 추가 | 지금 쓰는 MongoDB 안에 벡터 검색 켜기 |
| **Pinecone** | 대형 물류센터 임대 | 전문 회사가 다 해줌. 돈 냄. 무한 확장 |

## 2. 핵심 비교표

| 항목 | ChromaDB | MongoDB Atlas Vector Search | Pinecone |
|------|----------|------------------------------|----------|
| 형태 | 로컬 라이브러리/임베디드 | 클라우드 SaaS (Atlas 전용) | 클라우드 SaaS (서버리스) |
| 설치/시작 | `pip install` 즉시 | Atlas 계정+클러스터 생성 | 가입+API키 |

| 인터넷 필요 | X (오프라인 가능) | O | O |

## 3. 각각 자세히

### ChromaDB — 내 방 책장

**쓰는 법**
```python
pip install chromadb   # 끝. 설치 1줄, 서버 켤 필요 X
```

**장점**
- 💰 공짜. 카드 등록 X
- ⚡ 5분 시작. 초보자 최적
- 🔒 데이터가 내 PC/홈서버에만 → 감정·추모 민감 정보 외부 유출 X
- 인터넷 없어도 됨

**단점**
- 데이터 수십만~수백만 개로 커지면 느려짐 (단일 PC 한계)
- 서버 죽으면 직접 살려야 함

**우리한테**: MVP·개발 단계에 딱. 홈서버에 얹으면 됨.

---

### MongoDB Atlas Vector Search — 쓰던 창고에 칸 추가

**핵심**: 백엔드가 이미 MongoDB로 데이터(회원·반려동물 정보) 저장 중. 그 **같은 DB 안에서** 벡터 검색도 켜는 것.

```
[기존] MongoDB → 회원 정보, 반려동물 프로필
[추가] 같은 MongoDB → 위로 글 벡터도 저장 + 검색
       ⇒ DB 하나로 통합. 관리 포인트 1개
```

**장점**
- 📦 DB 하나로 통합 → 인프라 단순, 백업 한 번에
- 관리형(클라우드) → 서버 죽을 걱정 적음

**단점 (중요)**
- ⚠️ 이 기능은 **MongoDB Atlas(클라우드 유료)** 에서만 됨
- 홈서버 **로컬 MongoDB로는 안 됨**
- Atlas 쓰면 데이터가 클라우드로 나감 + 월 요금

**우리한테**: "지금 MongoDB가 Atlas냐 로컬이냐"가 갈림길. 로컬이면 이 옵션 **불가**.

---

### Pinecone — 대형 물류센터 임대

**쓰는 법**: 가입 → API 키 → 코드에서 그 회사 서버에 저장/검색 요청. 확장·속도 전부 그쪽 처리.

**장점**
- 🚀 수백만~수천만 개도 빠름. 자동 확장
- 운영 부담 거의 0

**단점**
- 💳 규모 커지면 유료 (무료 티어는 작음)
- ☁️ 데이터 외부 회사 서버 저장
- 지금 규모엔 과함

**우리한테**: 나중에 실서비스 트래픽 커지면 그때 재검토.

## 4. 결론 — 우리 상황 기준

```
지금 (MVP · 홈서버 · 민감 데이터 · 초보 팀)
   → ChromaDB ★ 추천
     공짜 + 5분 시작 + 데이터 로컬 보관

DB 하나로 합치고 싶다 + Atlas 전환 OK
   → MongoDB Atlas Vector Search
     (단, 로컬 MongoDB면 못 씀)

서비스 대박 + 사용자 폭증
   → Pinecone (그때 가서)
```
