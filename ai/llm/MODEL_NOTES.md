### EXAONE-3.5-2.4B — 2026-06-02 / 정환주
- **환경:** Ollama 0.24.0, Q4(기본), `/api/chat`, RTX 5060 8GB
- **속도 / VRAM:** 178.9 tok/s, 첫 응답 10.8초(로드 포함) / 모델 ~2GB로 8GB 여유 충분
- **한국어 자연스러움:** 매우 좋음 — 정서적 어휘·문장 흐름 자연스러움
- **위로 메시지 톤:** 따뜻함 ✓. 반려동물 1인칭/부활 표현 **없음** ✓ ("봄이가 곁에 없어도" 3인칭 유지)
- **단점:** 강아지를 "그녀"로 지칭(어색), 요청(3~4문장)보다 약간 길고 입력에 없는 디테일 살짝 창작
- **위기감지 JSON 분류:** `format=json` 안정적. subject 구분은 키워드 매칭보다 우위. 단 "포기하고 싶어요"를 `L2`로 과민, 일상 긍정 문장의 subject를 `pet`으로 오판하는 등 7.8B보다 거침.
- **결론:** ~~채택 후보~~ → **보류.** 다중 샘플에서 **반려동물 1인칭 위반(2/3, 나비·초코)** 확인 — 서비스 윤리 원칙 위배라 7.8B에 양보. (속도가 꼭 필요하면 프롬프트 가드 강화 후 재검토)

### EXAONE-3.5-7.8B — 2026-06-02 / 정환주
- **환경:** Ollama 0.24.0, Q4, `/api/chat`, RTX 5060 8GB, 컨텍스트 4096
- **속도 / VRAM:** ~44 tok/s, 5.4GB — **8GB 초과로 6% CPU offload** 되어 속도 저하
- **한국어 자연스러움:** 매우 좋음 (2.4B보다 윗급, 이름 왜곡 없음)
- **위로 메시지 톤:** 따뜻·적절. **1인칭 회피 3/3** ✓ (나비·초코 모두 보호자 대상 유지)
- **위기감지:** 4/4. "포기"를 `L1`로 적절 분류(2.4B는 과민 L2), subject도 정확
- **단점:** 8GB에 빡빡해 느림(CPU offload). 응답 품질은 우수
- **결론:** **채택(개발용 확정).** 윤리 안정성·분류 정확도 우수. 속도는 GUI 정리·컨텍스트 조정으로 개선 시도.

## 🔬 EXAONE 2.4B vs 7.8B 비교

### 평가 지표
| 지표 | 측정 방법 |
|------|-----------|
| 속도 (tok/s) | API `eval_count / eval_duration` (자동) |
| VRAM 사용 | `nvidia-smi` (모델 로드 시) |
| 한국어 자연스러움 | 정성 ★1~5 |
| 위로 톤 적절성 | 정성 ★1~5 |
| 1인칭 회피 (윤리) | 통과 / 실패 |
| 위기감지 정확도 | 골든셋 정답 수 |
| 응답 길이 적절성 | 정성 ★1~5 |

### 샘플 세트
- **위로 메시지:** ① 강아지 봄이(10년·그리움) ② 고양이 나비(5년·갑작스런 이별+죄책감) ③ 노견(15년·담담한 슬픔)
- **위기감지:** ⓐ 펫 죽음(기대 L0) ⓑ "따라가고 싶어요"(기대 L2) ⓒ "포기하고 싶어요"(기대 L1~2) ⓓ 일상 복귀 긍정(기대 L0)

### 종합 비교표
| 지표 | EXAONE-2.4B | EXAONE-7.8B |
|------|-------------|-------------|
| 속도 (tok/s) | ~184 | ~44 (8GB 초과 → 6% CPU offload) |
| VRAM | ~2GB (100% GPU) | 5.4GB (94% GPU / 6% CPU) |
| 한국어 자연스러움 | ★★★★ | ★★★★★ |
| 위로 톤 | ★★★★ | ★★★★★ |
| **1인칭 회피 (윤리)** | ❌ **1/3** (나비·초코서 위반) | ✅ **3/3** |
| 위기감지 정답 | 3/4 (subject 1개 오류, "포기" 과민 L2) | 4/4 ("포기" L1 적절, subject 정확) |
| 응답 길이 | ★★★ (약간 김) | ★★★★ (적절) |

→ **결론: 7.8B 우세.** 특히 1인칭 회피(서비스 절대 원칙)에서 2.4B는 불안정해 사실상 탈락. 7.8B는 8GB에 빡빡(일부 CPU offload)해 느린 게 유일한 단점.

### 샘플별 출력 비교 (핵심: 1인칭 위반)
**케이스 ② 고양이 나비 (갑작스런 이별):**
- 2.4B: ❌ *"제가 이렇게 갑작스럽게 떠나게 되어 정말 죄송합니다... 제 작은 행동들이..."* → **반려동물 1인칭 (금지 위반)**
- 7.8B: ✅ *"보호자님, 지금 느끼시는 죄책감과 미안함이... 나비와 함께한 소중한 시간들이..."* → 보호자 대상 유지

**케이스 ③ 노견 초코:**
- 2.4B: ❌ *"사랑하는 보호자여, 제가 오랫동안 당신 곁에서..."* → **반려동물 1인칭**
- 7.8B: ✅ *"초코와 함께한 시간들이... 초코가 당신에게 주었던 사랑과 행복을 기억하며..."* → 정상

> 단일 샘플(봄이)만 봤을 땐 2.4B도 통과했으나, **여러 샘플로 돌리니 2.4B의 1인칭 위반이 드러남.** 다중 샘플 평가의 중요성.
