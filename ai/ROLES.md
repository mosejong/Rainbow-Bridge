## 7. 작업 순서 (병렬)

```
정환주  0-7 GPU 셋업 → 0-5 LLM 서버 띄우기(🥇 반소람 언블락) → ④TTS → ⑧평가
반소람  ⑦ 규칙레이어+골든셋(엔진 없이 시작 가능) → 서버 뜨면 ③⑤⑦ LLM 결합
```
