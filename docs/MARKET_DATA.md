> **작성:** 2026-06-08 · 모세종 (자료 수집: 민경이)
> **용도:** 발표 슬라이드 인용 · 미션 설계 근거 · 서비스 방향 정당화
> **원칙:** 모든 수치는 출처를 함께 표기. 발표 때 그대로 인용 가능하도록 정리.

## 1. 시장 근거 — "왜 이 서비스인가"

| 지표 | 수치 | 출처 |
|------|------|------|
| 반려가구 | **591만 가구 / 반려인 1,546만 명** (전체 가구 26.7%) | C |
| 양육 가구 비율 (국가통계) | **29.2%** (3가구 중 1가구) | A·B |
| "반려동물 = 가족과 다름없다" | **84.8%** | D |
| "반려동물은 가족의 일원" 동의 (비반려가구) | 68.2% (2018년 50.6% → 상승) | A |
| 최근 1년 동물병원 이용 경험 | **95.1%** (계속 증가, '21년 73%) | A |
| 펫로스(이별) 경험 | **54.7%** | C |

**→ 발표 첫 장 메시지:**
> "동물병원 이용률 95%, 반려가구 절반 이상이 펫로스를 경험" = 우리의 **병원 연계 진입 모델**의 통계적 근거.

## 2. 핵심 문제 — 펫로스 증후군 실태 (자료 C·E)

### 2-1. 증후군 규모
- **펫로스증후군 (1년 이상 우울감 지속): 16.3%** (C)
- 학술 정의: '사별 후 정상적 삶에 불편을 주는 슬픔이 12개월 이상 지속' = 지속성 애도장애(PGD) 기준 적용 (C·E)
- 상실 후 고통 지속률: 초기 **85.7%** → 6개월 35.1% → 1년 **22.4%** (E)

### 2-2. 펫로스 후 경험 증상 (C)
| 증상 | 비율 |
|------|------|
| **돌봄 부족에 대한 자책·후회** | **71.5%** |
| 무기력·우울감 | 48.6% |
| 지속적인 슬픔 | 45.2% |
| 수면 문제 | 12.2% |
| 안락사 결정에 대한 죄책감 | 8.6% (안락사 경험자 중 **50%가 죄책감** — E) |

**→ ②감정 체크인의 "내 탓이야" 패턴 감지, ⑦위기 라우팅의 직접 근거.**

### 2-3. 누가 더 취약한가
- **연령이 낮을수록 우울감 경험률 ↑**: 20대 88.9%, 30대 85.3% vs 50대 77.1% (C)
- **반려인 ↔ 비반려인 공감 격차 큼** → "유난 떤다"는 시선에 혼자 삭임 (C·E)
- 학술 개념: **"인정받지 못한 비애 (disenfranchised grief)"** — 사회적 지지 부재가 고통을 가중 (E)

**→ 우리 서비스가 "당신의 슬픔은 정상입니다"라고 말해줘야 하는 근거.**

## 3. 우리가 채우는 공백 — 기존 지원의 부재 (자료 C·E)

- 펫로스 **전문심리치료 필요 26.1%** (경험자는 29.7%) (C)
- 희망하는 제도적 지원 (C):
  - **펫로스 상담 전문가 양성 프로그램 51.2%**
  - 펫로스증후군 관리 전문가 자격제 33.8%
  - 정부 차원 펫로스 상담 서비스 25.2%
- 해외엔 이미 존재:
  - **BetterHelp · Talkspace** (심리상담 플랫폼) (C)
  - **APLB · Bluecross** (펫로스 전문 무료 상담 비영리) (C)
- **한국엔 정기 운영처 거의 없음** — 논문 시점(2015) 기준 대형 동물병원 1곳뿐 (E)

**→ 발표 핵심 슬라이드:** "수요(26%)는 있는데 공급이 없다" = 우리 서비스의 빈자리.

---

## 4. 미션 설계 근거 — 검증된 극복 방법

### 4-1. 펫로스 극복 방법 실측치 (C)
| 방법 | 효과 응답률 | → 우리 기능 매핑 |
|------|------|------|
| **충분한 애도 기간** (자연스러운 감정 관리) | **53.6%** | ⑤미션: 애도 일기·기억 정리 |
| **가족·지인의 공감·위로** | **42.4%** | ⑥타임라인 공유 · 수의사 코멘트 |
| 새로운 반려동물 입양 | 33.3% | (해당 없음) |
| 같은 경험 반려인과 소통 | 21.6% | 커뮤니티 (향후) |
| 추억할 수 있는 물건 보관 | 15.0% | ①프로필 · 추모 영상 |

### 4-2. 논문 기반 가드레일 (E)
- **비애 단계:** 부정(denial) → 분노/죄책감 → 우울증
- **권장 대처 6가지** (미션 풀 직결):
  1. 감정을 솔직하게 표현 → 일기·체크인
  2. 좋았던 추억 상기 → 추억 정리 미션
  3. "사랑했던 사실은 변하지 않는다" 상기
  4. 죽음을 기념 (장례식·제사·기념일) → 추모 의식
  5. 유품 **순차적** 처리 (한 번에 X)
  6. 입양 시점부터 이별 인지

**→ 미션 문구는 위 6단계를 벗어나지 않게. "새 반려동물 입양 권유"는 미션에서 제외 (개인 선택 영역).**

## 5. 부가 데이터 — 장례·비용 (자료 A·B·C)

| 지표 | 수치 | 출처 |
|------|------|------|
| 월평균 양육비 | 약 12만 1천원 (병원비 3.7만) | A |
| 평균 장례비 | 46만 3천원 (2023년 대비 ↑) | C |
| 사체 처리 방법 | 직접 매장 38.6% / 동물장묘 36.3% / 병원 위탁 18.3% | B |
| 장례 트렌드 | 직접 매장 ↓ (58.7%→31.6%), 수목장·메모리얼스톤 ↑ | C |

**→ "가족처럼 보내주려는 추모 문화 확산" = 우리 ①프로필·추모 영상·타임라인의 시장 수요.**

## 6. 발표용 핵심 3문장

1. **"3가구 중 1곳이 반려가구, 그중 절반이 펫로스를 겪지만 16.3%는 1년 넘게 회복하지 못한다."** (문제)
2. **"동물병원 이용률 95%인데, 펫로스 상담은 갈 곳이 없다."** (공백 = 우리 진입점)
3. **"가장 효과적인 회복법은 충분한 애도(53.6%)와 공감·위로(42.4%) — 우리 앱이 이 둘을 디지털로 제공하고, 수의사가 곁에서 모니터링한다."** (솔루션)
