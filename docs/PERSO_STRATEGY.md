# PERSO AI 활용 전략

> **최종 수정:** 2026-06-05
> **담당:** 장민수 (멀티모달) · 모세종 (PM)

---

## 핵심 방향 (확정)

> **PERSO 아바타가 "반려동물"이 아니라 "추모 안내자"로 말하면,
> 기술 활용도 살고 윤리선도 지킨다.**

PERSO AI 아바타 = **추모 안내자 (Memorial Guide)**
반려동물 사진 = **LivePortrait용 추억 이미지** (별개 트랙)

---

## 1. 서비스 내 역할 분담

```
Gemini API        → 추모 메시지 생성 / 미션 추천 / 위기감지 보조
Google Cloud TTS  → 보호자 대상 편지 낭독 (warm / calm / soft 톤)
LivePortrait      → 반려동물 사진 기반 잔잔한 추억 영상 (별개 트랙)
PERSO AI 아바타   → 추모 안내자가 보호자에게 Gemini 메시지를 영상으로 낭독
```

---

## 2. PERSO 서비스 흐름

```
Gemini 추모 메시지 생성
↓
사용자가 전달 방식 선택 (선택형 — 기본값 아님)
   ① 텍스트로 보기
   ② 음성으로 듣기 (Google TTS)
   ③ AI 추모 안내자 영상으로 보기 ← PERSO
↓
PERSO 아바타가 메시지를 낭독한 MP4 생성
↓
타임라인에 "오늘의 추모 영상"으로 저장
```

> PERSO는 **선택 기능**입니다. 메인 필수 흐름에 포함하지 않습니다.
> (API 대기시간으로 인한 시연 리스크 방지)

---

## 3. 반려동물 성격 → 낭독 톤 연결

반려동물 성격 정보를 바탕으로 **추모 안내자의 낭독 톤을 추천**합니다.
"반려동물 목소리 복원"이 아닙니다.

| 반려동물 성격 | 아바타 발화 톤 |
|-------------|-------------|
| 발랄함       | 밝고 따뜻한 톤 |
| 애교 많음    | 부드럽고 다정한 톤 |
| 성숙함       | 차분하고 낮은 톤 |
| 듬직함       | 안정적이고 담담한 톤 |

```
❌ "우리 아이가 발랄했기 때문에 아이의 목소리를 복원했습니다"
✅ "보호자가 기억하는 아이의 분위기에 맞춰, 추모 안내자의 낭독 톤을 조정했습니다"
```

## 4. 화면 문구 (확정)

```
AI 추모 안내자 영상 만들기

보호자가 남긴 추억을 바탕으로 생성된 추모 메시지를
AI 안내자가 차분히 읽어드립니다.

※ 반려동물의 목소리나 모습을 복원하는 기능이 아닙니다.
```

---

## 5. 윤리 원칙

**고정 원칙:**

- PERSO 아바타는 **추모 안내자** 역할입니다. 반려동물을 흉내 내지 않습니다.
- 결과물에는 "AI 추모 안내자가 낭독한 기억 기반 추모 영상" 라벨을 표시합니다.
- 사용자가 명시적으로 선택한 경우에만 생성합니다.
- 위기 감정 감지(risk_level 2+) 시 PERSO 영상보다 **1393 안내 우선**.

**절대 금지:**

- PERSO 아바타를 반려동물인 것처럼 연출하거나 설명하는 것
- 반려동물이 직접 말하는 영상 생성
- 사용자 선택 없이 자동 생성

---

## 6. 구현 구조 (비동기)

```
POST /api/v1/media/{asset_id}/perso        # 후처리 작업 요청
GET  /api/v1/media/{asset_id}/perso/status # 작업 상태 조회

상태값: pending → processing → completed / failed
```

시연에서는 사전 생성된 샘플로 `completed` 상태를 보여줍니다.

---

## 7. 발표 포지션

```
레인보우 브릿지는 반려동물을 재현하거나 대신 말하게 하지 않습니다.

대신 PERSO AI 아바타를 '추모 안내자'로 활용해,
Gemini가 생성한 기억 기반 추모 메시지를 보호자에게 영상 형태로 전달합니다.

이를 통해 텍스트와 음성만 제공하는 것이 아니라,
보호자가 저장하고 다시 볼 수 있는 추모 영상 콘텐츠로 확장했습니다.
```
