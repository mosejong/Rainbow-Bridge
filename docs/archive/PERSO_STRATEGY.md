## 1. PERSO 기능 범위 (확정)

| 기능 | PERSO 가능? |
|------|------------|
| 텍스트 생성 (LLM) | ❌ 엔드포인트 없음 |
| 단독 TTS | ❌ 더빙 프로젝트 내부 전용 |
| 영상 더빙·번역 | ✅ (단, 동물 얼굴 불가) |
| 립싱크 (동물 영상) | ❌ **구조적 불가 — 드랍 확정** |
| STT (자막 추출) | ✅ |

**PERSO 립싱크 드랍 이유**: 방법 2 실험에서 얼굴 크롭 전처리 후에도 동물 얼굴 감지 실패 확인.
자연스러움 1/5, 입 움직임 없음. 구조적으로 사람 얼굴 전용 모델이라 동물에 적용 불가.

## 2. 확정된 서비스 파이프라인

```
Gemini API        → 추모 편지 생성 / 미션 추천 / 위기감지 보조
ElevenLabs TTS    → 반려동물 성격 기반 목소리 선택 + 편지 낭독
LivePortrait      → 반려동물 사진 → calm 드라이빙 영상 기반 추모 영상
FFmpeg            → 영상 + TTS 오디오 합치기
```

> Google Cloud TTS는 ElevenLabs 장애 시 폴백으로만 유지.

## 4. TTS — ElevenLabs 방향

반려동물 성격에 따라 목소리를 매핑:

| 성격 | 목소리 방향 |
|------|------------|
| 활발·장난꾸러기 (말티즈 등) | 밝고 통통 튀는 목소리 |
| 순둥이·온화 (골든리트리버 등) | 따뜻하고 포근한 목소리 |
| 도도·독립적 (고양이 등) | 차분하고 낮은 목소리 |
| 노령·노견 | 느리고 온화한 목소리 |

ElevenLabs는 stability/similarity/style 파라미터로 감정 강도 조절 가능.
무료 플랜 월 10,000자 → 테스트 가능.

> 상세 엔진 비교 → [ai/tts/ENGINE_NOTES.md](../ai/tts/ENGINE_NOTES.md)

## 6. 시연 시나리오 (업데이트)

```
1. 보호자가 반려동물 사진 업로드 + 성격 태그 선택
2. LivePortrait → calm 드라이빙 영상으로 추모 영상 생성
3. ElevenLabs TTS → 성격 매핑 목소리로 추모 편지 낭독 mp3 생성
4. FFmpeg → 영상 + 오디오 합치기
5. 완성된 추모 영상 재생 · MP4 다운로드
```

## 7. 발표 멘트 (업데이트)

> PERSO AI는 동물 얼굴 감지 구조 한계로 립싱크 적용이 불가하여,
> LivePortrait 기반 추모 영상으로 방향을 전환했습니다.
> 사람의 발화 패턴을 반려동물 사진에 전이하는 방식으로,
> 과장 없이 조용히 살아 움직이는 추모 영상을 생성합니다.
>
> 음성은 ElevenLabs TTS를 활용해 반려동물 성격에 맞는 목소리를 선택하고,
> Gemini가 생성한 추모 편지를 낭독합니다.
> 반려동물이 실제로 말한다고 주장하지 않으며,
> 보호자가 남긴 추억을 AI가 재해석한 작별 영상임을 화면에 명시합니다.
