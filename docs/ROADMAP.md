## 🎯 발표 시연 시나리오 (초안)

```
1. 수의사(병원 웹) 로그인
   → 보호자 반려동물 증상 기록 확인
   → 수의사 조언 한 줄 남기기

2. 보호자(앱) 로그인
   → 반려동물 프로필 등록
   → 감정 체크인
   → 추모 메시지 생성 (RAG + Gemini)
   → TTS 음성 재생
   → 사진 업로드 → LivePortrait 영상 생성

3. 펫로스 케어
   → 미션 추천
   → 기념일 알림 (D+30)
   → 위기 감지 시 1393 안내
```
