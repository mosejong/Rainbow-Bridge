## ❗ 결정 필요 안건 (오늘 정해야 함)

### 1. L2(경고) 동작 — 문서 ≠ 코드 🚨
- **문서(L0~L3 응답 예시)**: L2는 위로 메시지 **생성** + 1393 첨부, `source=local`
- **실제 코드**: `HOTLINE_REQUIRED_FROM=L2` → L2부터 생성 **차단**, content=1393, `source=safety` (L3와 동일)
- **결정:** L2에서 위로 메시지를 생성할 것인가(문서대로 코드 수정) vs 차단 유지(문서 정정)? → **반소람 영역**

### 2. "말하는 동물 영상" 핵심 기능 승격 여부
- 튜터님 *"사진 넣으면 그 아이가 말하는 영상 = 추억의 핵심"* 방향
- 현재 스펙상 **가산점(멀티모달)** 위치 → 핵심 기능으로 올릴지 결정

### 3. 미구현 백로그 2건 (반소람 지적)
- **vet_protocol RAG** — triage.py에 RAG 미연결. "증상 triage에 수의학 RAG 연결" 결정은 났으나 미구현
- **유저 편지→AI 답장** — 회복 절대점수 게이트 + 편지 위기 선체크, 설계 합의·미구현

### 4. welfare_resources(L1 상담자원 9건)
