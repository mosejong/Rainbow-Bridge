## 🔌 PERSO API 테스트 결과

| 기능 | 가능 여부 | 비고 |
|------|-----------|------|
| 영상 더빙·번역 | ✅ | 영상 업로드 → 다국어 더빙 |
| 립싱크 | ✅ | 영상 프로젝트에 입모양 맞춤 |
| STT (자막 추출) | ✅ | 영상/오디오 → 텍스트 |
| 범용 TTS | ❌ | 더빙 프로젝트 내부 전용, 단독 호출 불가 |
| 텍스트 생성(LLM) | ❌ | 엔드포인트 없음 |
| 정지 이미지 → 말하는 영상 | ❌ | 립싱크는 영상 입력 전용 |

**결론:** 추모 메시지 생성 → Gemini, TTS → Google Cloud TTS로 대체 완료.
PERSO는 멀티모달(장민수) 영상 더빙 파트에서 활용 검토 예정.
