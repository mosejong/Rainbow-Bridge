## ❓ 논의 안건

### 1. 웹 vs 앱

**현재 방향:** React(웹) — 민경이·장민수님이 Vite + React로 계획 중

**논의 포인트:**
- 웹 서버에 모델 올려서 속도·접근성 개선 가능한지
- 클라우드 서빙 vs 로컬 GPU 서버(정환주) 중 어느 쪽이 시연에 유리한지
- 모바일 앱(React Native 등) 전환 시 일정상 가능한지
- 평가 기준에서 웹/앱 중 선호하는 형태가 있는지

**강사님께 질문:**
> 현재 웹(React) 기반으로 진행 중입니다. 시연·평가 관점에서 웹과 앱 중 권장 방향이 있나요? 또한 LLM 모델을 웹 서버에 올려 속도를 개선하는 방향이 적절한지 의견 부탁드립니다.

### 2. 위기 감지 점수 체계 합의

**현황:** 반소람님이 L0/L1/L2 점수 체계 설계 완료
- L0 (정상, 0~15점) → 안내 없음
- L1 (우려, 16~49점) → 부드러운 안내 + 1393 언급
- L2 (경고, 50점+) → 즉시 1393 안내 모달

**결정 필요:** 백엔드 스키마 `risk_flag(bool)` → `risk_level(L0/L1/L2)` + `risk_score(int)` 변경 합의
