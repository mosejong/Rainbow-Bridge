## 🎯 확정된 서비스 흐름

**진입점:** 수의사가 시한부 선고 직후 앱 추천 → 보호자 설치

```
① 시한부  → 앱 설치 · 프로필 등록
             버킷리스트 / 사진 기록 / 일기  ← 남은 시간 채우기 (반복)
             [사이드탭] 증상 안내 (필요할 때)

② 죽음    → 이별 발생
             이별 직후 처치 안내 (담요, 닦기, 눈 감기)
             놀라지 마세요 안내 (사후 경직 등)

③ 장례    → 장례 준비 안내
             주변 동물 장례식장 (카카오 API)

④ 치료    → 감정 체크인 → 추모 메시지(생전 기록 반영) → TTS → 영상 → 회복 미션
```

**온보딩 카피 (확정):** "아이가 강아지별로 이사를 준비하고 있어요. 남은 시간동안 가서도 행복하게 좋은 추억 만들어 보아요."

> ❌ 수의사 웹 — 드랍. 우리는 수의사가 아닙니다. 증상 안내는 고정 텍스트로만.

### 2. 증상 안내 (사이드탭 / 별도 메뉴, 신규)
- **메인 흐름에 넣지 않음** — 보호자가 필요할 때 꺼내보는 참고 정보

- 톤: "아이가 힘들어할 수 있어요, 곁에 있어주세요" — AI 생성 아님, 고정 텍스트

### 3. 사진 기록 화면 (신규 또는 타임라인 연계)
- 사진 업로드 + 한 줄 메모

### 4. 일기 화면 (신규)
- 날짜별 짧은 일기 작성
