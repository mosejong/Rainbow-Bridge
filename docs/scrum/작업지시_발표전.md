```
시한부 선고 → 앱 설치 · 프로필 등록
  → 버킷리스트              ← 남은 시간 동안 함께 하고 싶은 것
  → 사진 기록               ← 함께한 순간들

  → 감정 체크인 → 추모 메시지 + TTS → 추모 영상 → 회복 미션

  [사이드탭 / 별도 메뉴]
  → 증상 안내 (고정 텍스트)  ← 필요할 때 꺼내보는 참고 정보
```

## 👤 민경이 (프론트엔드)

> 수의사 웹은 하지 않습니다. 아래 신규 화면 4개가 우선입니다.

### 1. 버킷리스트 화면 (신규, 메인 흐름)
- 항목 추가/체크 UI
- 예시 항목 미리 채워두기: "함께 산책하기", "좋아하는 간식 먹기", "사진 찍기"

### 3. 증상 안내 (사이드탭 / 별도 메뉴, 신규)
- **메인 흐름에 넣지 않음** — 보호자가 필요할 때 꺼내보는 참고 정보
- 고정 텍스트 카드 2단계:
  - **초기~중기:** 식욕 부진, 활동량 감소, 은둔 또는 집착
  - **말기~임종 직전:** 거친 호흡, 사지 냉증, 반응 저하
- 톤: "아이가 힘들어할 수 있어요, 곁에 있어주세요" — AI 생성 아님, 고정 텍스트

### 4. 사진 기록 화면 (신규 또는 타임라인 연계)
- 사진 업로드 + 한 줄 메모
- 기존 타임라인과 연계 가능

### 5. 일기 화면 (신규)
- 날짜별 짧은 일기 작성
- 기존 추모 타임라인과 연계 가능

### 6. E2E 흐름 최종 확인
- 새 흐름: 로그인 → 프로필 → 버킷리스트 → 사진·일기 → 감정 체크인 → 메시지 → TTS → 영상 → 미션
- 증상 안내는 사이드탭에서 접근 가능하게
- SafetyModal: `message.jsx`, `emotion.jsx` 에서 실제로 뜨는지 직접 확인
