- [x] **페르소나 카드 작성**
  - 마크: 반려동물 보호자 (병원·장례 채널 통해 유입)
  - 가치관: 슬픔을 존중하되 일상 복귀를 돕는다
  - 지원 범위: 기억 기반 추모 메시지, TTS 낭독, 미션, 감정 체크인
  - 금기 사항: 반려동물 부활·1인칭 사칭 금지 / 위기 시 1393 연결
  - 상세 → [SERVICE_FRAME.md](../SERVICE_FRAME.md)

- [x] **타깃 사용자 및 핵심 시나리오 3개**
  1. **시한부 진입** — 수의사에게 시한부 선고 받은 직후, 보호자가 버킷리스트·추억 기록 시작
  2. **이별 직후** — 임종 후 혼자 나오는 "공백 시간", 장례 안내 + 위로 메시지로 감정 지지
  3. **회복 단계** — 3회 이상 감정 체크인 후 회복 게이트 통과 → 일상 복귀 미션·1인칭 추모 편지 해금

- [x] **기존 서비스 대비 차별점**
  - 단독 앱이 아닌 **B2B2C 연계형** (병원·장례업체가 채널, 보호자 무료)
  - 병원→장례 "공백" 시간을 직접 타깃으로 하는 서비스
  - 회복 게이트(`content_unlocked`) — 위기 상태 보호자에게 추모 콘텐츠 차단
  - 위기 감지 4단계(L0~L3) + 1393 자동 라우팅

- [x] **결과물 형태 및 모달리티 선택 확정**
  - 텍스트: LLM 기반 추모 메시지 (Gemini 2.5-flash)
  - 음성: Google Cloud TTS (한국어 Neural2)
  - 영상: LivePortrait 사진→영상, FFmpeg merge_audio (faststart)
