💡 왜 필요한가

펫로스 초기에 반려동물의 목소리·편지를 바로 보여주면
집착·현실 부정으로 이어질 수 있어요.

회복이 어느 정도 된 시점에 제공해야 치유 효과가 생깁니다.

→ "회복하면 OO이의 편지를 받을 수 있어요"

📊 콘텐츠 단계

0~49
초기·위기 단계
위로 메시지, 미션 추천, 타임라인 편지 잠김
50~79
회복 중
위 전체 + 찌라시 카드 노출 편지 예고만
80+
회복 완료
기억 기반 추모 편지 (최초 1회 무료) + LivePortrait 영상 무료 생성 추가는 유료

🔑 1회 영구 언락 구조

점수를 올렸다 내렸다 해서 편지를 반복 수령하는 악용 차단

🎉 최초 80점 달성
편지 1회 무료 생성 → DB 영구 저장. 이후 점수 변동 무관하게 언제든 다시 볼 수 있음 (토큰 소비 없음)
📉 점수 하락 후 재상승
기존 편지 조회만 가능. 새 생성 없음
✍️ 새 편지 재생성 요청
크레딧 차감 — 우리도 토큰 비용 발생하므로 유료

🩺 편지 열람 후 7일 모니터링

편지를 받은 뒤 오히려 감정이 떨어질 수 있다는 피드백 반영

편지 열람
    │
    ▼
7일 집중 모니터링 시작
    │
    ├─ 점수 유지·상승 → 정상 흐름 ✓
    └─ 점수 -20 이상 하락 → 위로 강화 + 미션 재조정
                          └─ risk_level 2+ → 1393 즉시 🚨

⚠️ 윤리 원칙

  • 점수가 낮아도 강제하지 않는다 — 보호자 페이스 존중
  • 80점이 "완치"를 의미하지 않는다 — "충분히 준비됐어요" 표현 사용
  • risk_level 2+ 상태에서 언락·재생성 없음
  • 언락 후 면책 문구: "AI가 보호자가 전해준 추억을 바탕으로 재해석한 편지입니다"
  • 1인칭 화법은 80점+ AND risk_level 0~1 동시 충족 시에만 허용 (강사 승인 기준)
  • 편지 열람 후 7일 집중 모니터링 — 재우울 조기 감지
