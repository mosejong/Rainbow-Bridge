# 디자인 & 기능 이슈 트래킹

> 해결되면 상태를 `✅ 수정 완료`로 바꾸고 해결 방법 간단히 기록해주세요.

---

## 화면 / 디자인

- **온보딩 화면 Android 미적용**
  아이폰에서는 디자인이 적용되는데 안드로이드에서는 안 보임.
  화면 작동은 되지만 흰색 하단이 잘려있고, 앞에 하얀 화면 표시됨.

- **이중 탭 노출** (오늘의 미션, 회복 리포트) `✅ 수정 완료`
  탭 위에 '뒤로', '홈', '로그아웃'이 이중으로 표시됨.
  → `_layout.jsx`에서 mission·report에 `headerShown: false` 추가 (2026-06-12)

- **추모 편지 배경 플랫폼별 차이**
  아이폰과 안드로이드에서 추모 편지 배경 색 및 디자인이 다르게 보임.

- **안드로이드 추모 편지 글씨 색 오류**
  추모 편지 하단 '열어볼까요?' 텍스트가 안드로이드에서만 흰색으로 표시됨.

---

## 입력 / UX

- **버킷리스트 키보드 가림** `✅ 수정 완료`
  버킷리스트 항목 작성 시 자판이 올라오면 입력 내용이 가려져 안 보임.
  → `KeyboardAvoidingView behavior:'height'` + `scrollToEnd` 추가 (2026-06-12)

- **감정 체크인 후 화면 이동 오류** `✅ 수정 완료`
  감정 체크인 완료 후 오늘의 추모 편지가 아닌 오늘의 미션으로 넘어가는 문제.

---

## TTS / BGM

- **추모 편지 TTS 미재생** (아이폰·안드로이드 공통)
  BGM만 나오고 TTS 음성이 재생되지 않음.

- **음성으로 듣기 TTS 미재생** (아이폰·안드로이드 공통)
  목소리 변경해도 TTS가 재생되지 않고 '음성을 생성하고 있어요' 로딩만 계속됨.

- **BGM 화면 이탈 후에도 계속 재생** `✅ 수정 완료`
  추모 편지에서 홈으로 이동해도 배경음악이 계속 재생됨.
  → `useFocusEffect`로 blur 시 BGM/TTS stop 추가 (2026-06-12)

- **추모 편지 스크롤 불가** `✅ 수정됨`
  터치로 편지를 올려도 스크롤이 되지 않던 문제.

---

## 기능

- **추모 영상 생성 안됨**
  추모 영상 만들기가 동작하지 않음.

- **조사 처리 안됨**
  반려동물 이름에 따른 조사(이/가, 을/를 등) 자동 처리가 되지 않음.

- **사진 기록 최적 사진 자동 선택**
  로그인 후 더보기 → 사진 기록에서 여러 장 입력 시, 눈·코·입이 가장 잘 나온 사진을 자동으로 선별하는 기능 필요.
